## Overview
This repo contains standalone examples and explanations for essential JavaScript topics, including variable scoping, control flow, DOM manipulation, event handling, array methods, and string operations. Each topic is covered in its own file with clear, commented code that you can run and modify.

## Topics Covered

| Topic | Description |
|-------|-------------|
| **Variable Scope** | `var`, `let`, `const`; global, function, and block scope; hoisting behaviour. |
| **Flow Control** | `if...else`, `switch`, ternary operator; truthy/falsy values; conditional logic. |
| **DOM Manipulation** | Selecting elements (`getElementById`, `querySelector`), changing content & styles, creating/removing elements. |
| **Event Listeners** | `addEventListener`, common events (click, input, submit), event propagation, delegation. |
| **Array Methods** | `map`, `filter`, `reduce`, `forEach`, `find`, `some`, `every`, sorting, spreading. |
| **String Manipulation** | Template literals, common methods (`slice`, `split`, `replace`, `toUpperCase`), string interpolation. |
